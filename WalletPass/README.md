Starter Wallet pass source for `TallaLoyalty.pkpass`.

Files:
- `TallaLoyalty.pass/pass.json`: pass payload and fields
- generated images and signature files are created during packaging

This starter pass is signed and bundled with the app as a generic loyalty card.
To personalize it per customer later, regenerate the pass with customer-specific:
- serial number
- member name
- points balance
- tier
- barcode message
